Highly Available & Secure Cloud Architecture on Amazon Web Services

This project demonstrates a modern enterprise-grade AWS infrastructure built using cloud-native services and industry best practices.

The environment is designed for scalability, high availability, fault tolerance, and security using a multi-tier architecture.

🌍 Networking

Amazon VPC, Public & Private Subnets, Internet Gateway, NAT Gateway.

⚖️ Load Balancing

Application Load Balancer distributes traffic across multiple EC2 instances.

💻 Compute

Amazon EC2 Auto Scaling ensures elasticity and fault tolerance.

🗄 Database

Amazon RDS deployed in Multi-AZ configuration for high availability.

🔒 Security

IAM Roles, Security Groups, NACLs and least privilege access.
