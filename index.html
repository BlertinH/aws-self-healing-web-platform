Scalable Cloud-Native Architecture on Amazon Web Services

This project demonstrates a production-like AWS architecture designed for high availability, fault tolerance, and secure network segmentation using a multi-tier approach.

Traffic is distributed through an Application Load Balancer (ALB), which routes requests to healthy EC2 instances deployed inside private subnets, ensuring zero direct public access to backend services.

Architecture Overview

Internet → ALB → Private Subnet (EC2) → Application Layer

If you can see this page, your request successfully passed through AWS Load Balancer and reached a healthy backend instance.
Built as part of a Master’s thesis focused on Cloud Computing, DevOps practices, and Infrastructure as Code (IaC).
